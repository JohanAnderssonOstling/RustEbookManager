# Rust Ebook Manager

This is a simple ebook and pdf manager written in Rust and C++.

## Dependencies

### Qt 6.4.2
- Core
- Gui
- Quick
- WebView
- QuickControls2
- Pdf

### Corrosion

### Rust

### CMake 3.16

## Building

To build the project, run the following commands:

```bash
mkdir build
cd build
cmake -G "Unix Makefiles" ../ 
make
```

## Running

To run the project, run: 

```bash
./EbookTest
```
